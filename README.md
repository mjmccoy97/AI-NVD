 # AI NVD with a two-stripe rail-optimized design  
 # WORK IN PROGRESS


This AI NVD digital twin deployment is a two-stripe rail-optimized design with eight leafs per stripe and two spines acting as inter-stripe connectors. The topology, shown below, is deployed/instantiated using Containerlab 

![two-stripe-rail-optimized](/static/images/two-stripe-rail-optimized.png)

This AI NVD covers the following features:

- IPv6 fabric connectivity
- Leaf-GPU IPv6 connectivity
- Dynamic Load Balancing (DLB)
- DCQCN (ECN + PFC) for congestion control
